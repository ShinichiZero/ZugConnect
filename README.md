# ZugConnect

ZugConnect is an unofficial train departure assistant for Germany. It focuses on clear, accessible departure boards and step-by-step planning help. This project is open source and provided as-is.

## Live site

- https://shinichizero.github.io/ZugConnect/

## Highlights

- Fast station search with upcoming departures
- Full stop lists with platform details
- Step-by-step guide for travelers who need structured planning
- Accessibility improvements aligned with BITV 2.0 / WCAG

## Accessibility focus

- Keyboard-first navigation (skip link, clear focus states)
- Screen-reader friendly structure and announcements
- Reduced motion support
- Large tap targets and predictable controls

## Data sources

- Departures and stop times from Transitous (unofficial)
- This project is not affiliated with Deutsche Bahn

## Development
